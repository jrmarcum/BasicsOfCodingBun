#### To run the program, make sure you are in the program's folder, then use the Run Command below.
___
##### Run Command:

`$ node methods.js`

##### Results:

`area: 50`
`perim: 30`

#### Note: JavaScript methods are defined on classes (or object prototypes). There is no distinction between pointer receivers and value receivers as in Go — all method calls in JavaScript go through the object reference.
