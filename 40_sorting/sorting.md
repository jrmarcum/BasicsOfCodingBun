___
#### To run the program, make sure you are in the program's folder, then use the Run Command below.
___
##### Run Command:

`$ node sorting.js`

##### Results:

`Strings: [ 'a', 'b', 'c' ]`
`Ints:    [ 2, 4, 7 ]`
`Sorted:  true`
___

#### Note: JavaScript's `Array.prototype.sort()` sorts strings lexicographically by default. A numeric comparator `(a, b) => a - b` is required for correct integer sorting. Node.js displays arrays with comma-separated quoted elements rather than Go's space-separated unquoted format (e.g., `[ 'a', 'b', 'c' ]` vs `[a b c]`).
