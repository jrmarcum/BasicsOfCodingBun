___
#### To run the program, make sure you are in the program's folder, then use the Run Command below.
___
##### Run Command:

`$ node switch.js`

##### Results:

`write 2 as two`
`It's a weekday`
`It's after noon`
`bool`
`int`
`unknown type string`
___

#### Note: The day-of-week and time-of-day lines will vary with each run. JavaScript's `typeof` operator returns `"number"` for all numbers (both integers and floats), unlike Go's type switch which distinguishes `int` from `float64`.
