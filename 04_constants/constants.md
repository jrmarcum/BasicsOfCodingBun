___
#### To run the program, make sure you are in the program's folder, then use the Run Command below.
___
##### Run Command:

`$ node constants.js`

##### Results:

`constant`
`600000000000`
`600000000000`
`-0.28470407323754404`
___

#### Note: JavaScript's `const` declares an immutable binding, but unlike Go's untyped constants, JavaScript values are dynamically typed. JavaScript uses full decimal notation for 6e+11 (600000000000) while Go's `fmt.Println` uses scientific notation (6e+11).
